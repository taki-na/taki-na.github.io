x-ui 面板 VMESS/VLESS+WS+TLS+CDN 节点搭建

发表于2022-09-07|更新于2022-11-21|教程
|字数总计:376|阅读时长:1分钟|阅读量:

注意，本行为可能违反 cloudflare 自助订阅协议 第 2.8 节，请酌情使用，对于产生的后果本站不负任何责任

准备

  • VPS 一台
  • 域名一个
  • Cloudflare 账号一个

安装 x-ui

参考官方文档 https://github.com/vaxilu/x-ui

解析域名

进入 Cloudflare，选择你的域名 -> DNS -> 添加记录

记得打开代理

申请证书

可以使用脚本或宝塔面板申请证书，这里使用的是 cf 的免费证书

提示：cloudflare 颁发的免费证书只有 cloudflare 信任

进入 Cloudflare，选择你的域名 -> SSL -> 源服务器 -> 创建证书

复制源证书和私钥备用

搭建节点

  1. 进入 x-ui 面板新建节点
  2. 协议选 VMESS 或 VLESS
  3. 端口填 cf 支持的端口
  4. 传输选 WS
  5. 新建一个请求头，值填前面解析的域名
  6. 打开 tls，域名同上，公钥填源证书，密钥填私钥
